# Frontend SigaTCC

Interface web do sistema de gestão de TCCs, desenvolvida com Vue.js.

## Tecnologias

- Vue 3
- Vite
- Vue Router
- Axios

## Como executar

Instale as dependências:

```bash
npm install
```

Crie o arquivo `.env` com base no `.env.example`:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

Execute o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse:

```bash
http://localhost:5173
```

## Backend

O backend Django deve estar rodando em:

```bash
http://127.0.0.1:8000
```
